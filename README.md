# Processing-Noob-Masterpieces

主に大学の課題として作成したProcessing作品のアーカイブです。 
プログラミングを始めて間もない初心者の頃に作成したものが多いため、クオリティは大目に見ていただけますと幸いです。

※ 一部の作品（DVD_LOGO_Moving, EP_MusicPuzzle_Myidea_MakeWithAI）は、AIの補助を活用して作成しました。

---

## 🎨 作品紹介

### 💿 DVD_LOGO_Moving
- **概要**: DVDのロゴが跳ね回るスクリーンセーバー風の作品。
- **背景**: 「文字が動く」というお題の講義課題。AIの補助あり。

### 🧩 EP_MusicPuzzle_Myidea_MakeWithAI
- **概要**: 音源（wav/mp3）を視覚的・聴覚的にパズル化した音楽ゲーム。
- **モード**: 
  - **再現モード**: シャッフルされたピースを元の音源に戻す。
  - **創造モード**: ピースを自由に並べ替えて新しい音楽を作る。
- **背景**: AIの補助あり。

### 🌳 EP_RainAndTree
- **概要**: 雨が降り、木が成長するまでを描いたネイチャーアート。
- **背景**: 「植物」というお題に対し、習得した技術を総動員して作成。

### ❌ hw_MaruBatu
- **概要**: Processingの通信機能を用いた、2人対戦の〇✖ゲーム。
- **背景**: 大学の宿題として作成。

### 📰 hw_YahoonewsTopics
- **概要**: Yahoo!ニュースのRSS（XML）を読み込み、新着記事をリアルタイムで確認できるツール。
- **背景**: 大学の宿題として作成。。

---

## 🚀 実行方法

1. [Processing IDE (PDE)](https://processing.org/download/) をインストールしてください。
2. 各プロジェクトフォルダ内の `.pde` ファイルを開いて実行してください。
