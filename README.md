# Processing-Noob-Masterpieces

主に大学の課題として作成したProcessing作品のアーカイブです。
殆どはプログラミングを初めて間もない初心者の頃に作成したものなので、クオリティは大目に見てください。"DVD_LOGO_Moving","EP_MusicPuzzle_Myidea_MakeWithAI"はAIの補助ありで作成しました。

## 作品紹介
- **DVD_LOGO_Moving**: DVDのロゴが動くやつです。「文字が動く」というお題で講義時間内に自由に作成する課題にて作成した作品の一つです。AIの補助ありで作成しました。
- **EP_MusicPuzzle_Myidea_MakeWithAI**: wavやmp3ファイルを読み込んで、それをパズルにした音楽パズルです。シャッフルされたピースを元の音源に戻すモード（再現モード）と、シャッフルされたピースを自由に並び替えて新しい音楽を作るモード（創造モード）が搭載されています。AIの補助ありで作成しました。
- **EP_RainAndTree**:　雨が降って、木が生えるまでの一連を当時習ったことのみで自力で完成させた作品です。「植物」というお題で講義時間内に自由に作成する課題にて作成しました。
- **hw_MaruBatu**: Processingの通信を用いて、〇✖ゲームを作成しました。大学の宿題として作成しました。
- **hw_YahoonewsTopics**: 9つのトピックから気になるものをクリックすれば、そのトピック内の今現在出ているYahoo!ニュース新着記事を確認することができるものです。XMLファイルを読み込んでいます。大学の宿題として作成しました。

## 実行方法
Processing IDE (PDE) で各 .pde ファイルを開いて実行してください。
